# Code Happen
Code Crafters is a web platform that allows users to create, explore, and join technology events, both online and in person.
This repository contains the frontend of the project, built with React.js, providing a responsive and dynamic user interface connected to the Spring Boot API.

--- 

# How to excute
clone this repo in terminal
```
git clone https://github.com/TechMeetP9/CodeCrafters-Frontend.git
```
move to the repo that cloned

```
cd CodeCrafters-Frontend
```
then 
```
npm install
npm run dev
```
